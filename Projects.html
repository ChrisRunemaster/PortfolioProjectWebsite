Lucky Chan: A PHP Imageboard Simulator.

An Imageboard 'Ripoff' made up in PHP. The board is linked to a database that stores post content, including name, timestamp, comments, and any pictures. You can check it out here!

In addition, the board features built in security that doesn't allow posters to post html, scripts, or images with extra extensions. There are also randomly changing banners upon each refresh and a dynamic thumbnail generators for posted pictures.
